## üöÄ Getting Started

To run this project locally, follow these steps:

### Prerequisites

*   A modern web browser (Chrome, Firefox, Safari, Edge).
*   Python 3.x (recommended for local server) or any static file server.

### Installation & Running

1.  **Clone the repository**
    ```bash
    git clone https://github.com/tzolkowski96/global-population-insights.git
    cd global-population-insights
    ```

2.  **Start a local server**
    Because of modern browser security policies (CORS), it is best to run this project via a local server rather than opening the file directly.

    Using Python 3:
    ```bash
    python3 -m http.server 8000
    ```

3.  **View the site**
    Open your browser and navigate to:
    `http://localhost:8000`
